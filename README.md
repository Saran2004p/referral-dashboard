# Referral Dashboard

Referral management dashboard built with React.

## Features

- JWT Authentication
- Protected Routes
- Dashboard Overview
- Service Summary
- Referral Sharing
- Search Referrals
- Sort Referrals
- Client Side Pagination
- Referral Details
- Not Found Page

## Installation

npm install

## Run

npm run dev

## Build

npm run build
